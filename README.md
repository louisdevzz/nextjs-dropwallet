# Telegram Mini Wallet

### Installing package
```bash
npm install
```

### Setup env in .env.example
```bash
JWT_PINATA_CLOUD=
NEXT_PUBLIC_RELAYER_ACCOUNT_ID_NEAR_MAINNET=
RELAYER_PRIVATE_KEY_NEAR_MAINNET=
BLUNT_PRIVATE_KEY_MAINNET=
```
Convert .env.example to .env

### Setup API

* [API](./api/README.md)

### Run project

```bash
npm run dev
```
